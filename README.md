
# Weather.IO

Dashboard para clima, podendo ver temperatura, umidade, vento entre outras informações

## Tech Stack

**Client:** HTML, CSS, JavaScript, API

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Open with VSCode

```bash
  code .
```

Open with live server

```bash
  npm run start
```
